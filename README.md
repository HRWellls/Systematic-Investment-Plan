# 定投统计助手

一个简洁优雅的基金定投管理工具，帮助你轻松追踪和管理你的定投计划。

## 功能特点

### 定投概览统计
- **预计持仓总金额**：显示所有基金的目标投资总额
- **当前持仓金额**：实时展示当前已投资金额及完成进度
- **每月定投金额**：自动计算每月定投总额
- **基金数量统计**：一目了然掌握投资组合规模

### 基金分类管理
- 自定义基金分类（如：指数基金、债券基金、股票基金等）
- 支持为每个分类设置不同颜色标识
- 分类可编辑、可删除

### 基金记录管理
- 添加基金信息：名称、代码、目标金额、当前金额
- 设置定投计划：定投金额、定投频率（每日/每周/每两周/每月）
- 记录定投起始日期和备注信息
- 自动计算预计完成日期

### 未来计划记录
- 为每只基金记录未来定投调整计划
- 支持记录定投金额变更、频率调整等计划

### 数据管理
- **数据持久化**：所有数据自动保存到浏览器本地存储
- **导出备份**：支持将数据导出为 JSON 文件备份
- **导入恢复**：从备份文件恢复数据
- **清空数据**：一键清空所有记录（谨慎操作）

### 智能计算
- 自动计算每月定投总额
- 根据定投计划自动推算预计完成日期
- 支持按定投计划更新当前持仓金额并按照完成进度排序

## 使用方法

1. **添加分类**：在左侧"基金分类"区域添加你的基金分类
2. **添加基金**：点击右上角"+"按钮添加基金记录
3. **设置定投**：填写定投金额和频率，系统会自动计算预计完成日期
4. **记录计划**：点击基金卡片上的文档图标，记录未来调整计划
5. **数据备份**：定期使用"导出备份"功能保存数据

## 技术说明

- 纯前端实现，无需后端服务器
- 使用 LocalStorage 本地存储数据
- 响应式设计，支持移动端访问
- 模块化结构，代码组织清晰

## 项目结构

```
├── 定投助手.html        # 合成版HTML文件
├── index.html          # 主HTML文件
├── css/
│   └── styles.css      # 样式文件
├── js/
│   ├── utils.js        # 工具函数和常量
│   ├── storage.js      # 本地存储管理
│   ├── operations.js   # 业务逻辑和状态管理
│   └── render.js       # 渲染函数
├── README.md           # 项目说明
└── .gitignore
```

### 模块说明

- **utils.js**: 包含工具函数、常量定义和计算逻辑
- **storage.js**: 处理本地存储的读写操作
- **operations.js**: 实现核心业务逻辑和状态管理
- **render.js**: 负责页面渲染和用户交互

## 浏览器兼容性

- Chrome / Edge / Firefox / Safari 等现代浏览器
- 支持移动端浏览器

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会导致丢失
- 建议定期导出备份，以防数据丢失
- 更换浏览器或设备时需要手动导入备份数据
